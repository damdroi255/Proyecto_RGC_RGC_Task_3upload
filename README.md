INSTALL
	Ejecutar el Script de generación del Schemma de la BBDD en un gestor MySql/MaríaDB
	Habilitar un recurso en un servidor web que albergue la estractura de carpetas y archivos sumistrado.
	Asegurarse de que el fichero conexion.php contiene el usuario y password correspondiente al de la BBDD.
